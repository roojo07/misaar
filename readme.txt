	Небольшой сайт, содержащий информацию о товарах(пиломатериалы, вагонка, доска пола и т.д) с админской панелью.
	Написан с помощью ASP.Net MVC, jQuery, Bootstrap. База данных - MS SQL Server 2012. Доступ к бд осуществляется посредством Entity Framework. Подключена внешняя библиотека tinymce (текстовый редактор). Мобильная версия сделана с помощью jQuery-mobile.
	В админской панели реализованы стандартные методы управления содержимым базы данных (создание, редактирование, удаление, просмотр деталей). Схема базы данных представлена в файле MisaarDBDiagram.png
